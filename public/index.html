❓ Perguntas Frequentes
Como conecto minha conta Meta Ads? ▼
Vá em Integrações → Meta Ads, clique em "Conectar" e cole seu token de acesso. O token deve ter as permissões ads_read e ads_management. Após salvar, seus dados sincronizam automaticamente.
Meus dados são seguros? ▼
Sim. Todos os tokens são criptografados antes de serem salvos. Nunca compartilhamos seus dados com terceiros. A conexão com a Meta é somente de leitura e gestão.
Posso conectar mais de uma conta de anúncios? ▼
Sim! Você pode conectar quantas contas Meta Ads quiser e filtrar por conta na aba Campanhas.
Por que meu ROAS aparece como 0.00x? ▼
O ROAS é calculado com base na receita rastreada via UTM parameters. Certifique-se que seus anúncios têm UTMs e que o webhook de vendas está integrado.
Quando os produtos da loja estarão disponíveis? ▼
Os produtos estão sendo finalizados e serão lançados em breve. Deixe seu e-mail no topo desta página ou entre em contato pelo WhatsApp para garantir acesso antecipado com condições especiais.
Como funciona o programa de afiliados? ▼
Você ganha 30% de comissão recorrente por cada assinatura feita pelo seu link. Acesse a aba Afiliados para copiar seu link exclusivo.

📡 Meta CAPI — Rastreamento Server-Side
Configure seu Pixel e Token de Acesso do Meta para que a IPPMIFY envie eventos de Purchase server-side ao seu pixel toda vez que uma venda for registrada nos seus webhooks (Kirvano, Hotmart, Eduzz, Stripe etc.)
Encontre em: Meta Business Suite → Gerenciador de Eventos → seu Pixel → Configurações.
Gere em: Business Suite → Configurações → Usuários do sistema → Gerar token → marque ads_management. Salve em local seguro — não é exibido novamente.
🔒 O token é armazenado de forma segura e usado apenas para enviar eventos ao seu pixel. A IPPMIFY nunca compartilha suas credenciais.
🔗 CAPI Proxy — Rastreamento de Funil
Rastreie PageView, ViewContent, InitiateCheckout e Purchase server-side no seu Pixel via IPPMIFY. Cole o snippet JS nas páginas do seu funil — funciona em qualquer plataforma (ClickFunnels, WordPress, Hotmart Pages, Loja Integrada etc.)
Esta chave é pública — pode ficar visível no código do site. Rotacione se suspeitar de abuso.
Snippet JS — Cole no <head> de todas as páginas do funil: 

              
            


            
            
Exemplos de disparo no seu funil:

            
// Página de vendas (scroll ou tempo na página):
window.ippmify('track', 'ViewContent', { content_name: 'Curso X', value: 97.00, currency: 'BRL' });

// Clique no botão de compra / entrada no checkout:
window.ippmify('track', 'InitiateCheckout', { value: 97.00, currency: 'BRL' });

// Página de obrigado / confirmação de compra:
window.ippmify('track', 'Purchase', { value: 97.00, currency: 'BRL', order_id: '12345' });

// Capturar e-mail do lead (melhora o match rate no Meta):
window.ippmify('setUser', { email: 'lead@email.com', phone: '5511999999999' });
